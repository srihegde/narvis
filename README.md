# NARVis Project Website

This repository contains the source code and assets for the NARVis project page: https://srihegde.github.io/narvis/

**NARVis: Neural Accelerated Rendering for Real-Time Scientific Point Cloud Visualization**

Conference: **PacificVis 2026**

Paper: https://arxiv.org/pdf/2407.19097

Code: https://github.com/VALIXR-Neural-Rendering/narvis

## Project Summary

NARVis is a neural accelerated rendering approach for interactive visualization of large scientific point clouds. The project combines a high-performance multi-attribute compute rasterizer (MACR) with a neural post-processing network to emulate high-quality rendering effects while keeping runtime and memory practical for real-time use.
